## 评星组件：commentStars
    

### 组件属性
    
        /**
       * 组件的属性列表
       */
      properties: {
        maxStars:{
          type:Number,
          value:5
        },
    	stars:{
    		type:Number,
    		value:0
    	}
      },
   * maxStars:默认多5星
   * stars：评星数量
